# 封装

核心思想: 合理隐藏, 合理暴露

1. 如何隐藏: 使用private关键字修饰成员变量,这样就只能在本类中访问了
2. 如何暴露 : 使用public的get()和set()方法----取值和赋值

面向对象的三大特征:封装 继承 多态

# Javabean(实体类)

要求1: 满足封装(private成员变量 public的getter和setter方法)

要求2: 需要提供一个无参构造器, 有参构造器可选

只负责数据存取, 不考虑操作
